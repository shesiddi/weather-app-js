# Weather App

A modern and interactive weather application built using **JavaScript**, featuring real-time weather data, dynamic UI themes, and smooth animations.

---

# Live Demo

👉 https://your-username.github.io/weather-app

---

# Features

* Search the weather by city
* Real-time temperature, sunrise, and current time
* Dark / Light mode toggle (persistent)
* Interactive temperature chart (Chart.js)
* Smooth UI transitions and loader
* Responsive design

---

# Tech Stack

* HTML5
* CSS3 (Glassmorphism + Animations)
* JavaScript (Vanilla JS)
* OpenWeather API
* Chart.js
* Font Awesome

---

# Setup Instructions

1. Clone the repository

git clone https://github.com/your-username/weather-app.git
